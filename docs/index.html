BSC SMART Standards — Data Catalogue Package

Machine-readable JSON Schemas supporting the Digital Rulebook initiative. This repository demonstrates progression from SMART Level 2 (structured digital artefacts) towards Level 3 (machine-readable, validation-ready rule components).

Purpose
This package provides structured JSON Schema representations of selected BSC artefacts to support interoperability, validation, and future automation.

Scope currently includes:
  • CVA Annex B
  • BMRS
  • SVA Volume 2
These schemas form part of the broader transition toward a machine-readable Digital Rulebook.
